Frequently Asked Questions

Learn how to set up and configure your trading bots for optimal performance. Start by logging into your dashboard, selecting "My Bots," and following the step-by-step wizard. If you encounter errors, check your API keys in the settings.
Troubleshoot API connection issues with Deriv and other platforms. Ensure your API token is active and has the necessary permissions. Test the connection in the "Integrations" tab of your account settings.

Secure your account and troubleshoot login issues. Enable 2FA in settings, use strong passwords, and monitor active sessions. If locked out, use the "Forgot Password" link.
